## 🔍 常见问题

### Q: 如何添加新页面？

A: 查看 [添加页面指南](./添加页面指南.md)，按照步骤操作即可。

### Q: 路由是怎么生成的？

A: 项目使用 Elegant Router 文件路由系统，根据 `src/views/` 目录下的文件自动生成路由。运行 `pnpm gen-route` 即可生成。

### Q: 如何调用后端 API？

A: 在 `src/service/api/` 下定义 API 接口，然后在组件中导入使用。参考 [添加页面指南](./添加页面指南.md) 中的 API 部分。

### Q: 如何添加国际化文本？

A: 在 `src/locales/langs/zh-cn.ts` 和 `src/locales/langs/en-us.ts` 中添加文本，然后在组件中使用 `$t()` 函数。

### Q: 静态路由和动态路由的区别？

A: 
- **静态路由**: 前端文件路由，通过文件路径自动生成，适合固定菜单
- **动态路由**: 后端返回菜单配置，前端动态注册，适合权限控制的菜单
