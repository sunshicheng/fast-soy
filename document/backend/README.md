## 🔍 常见问题

### Q: 如何添加新功能？

A: 查看 [添加新功能指南](./添加新功能指南.md)，按照步骤操作即可。

### Q: 项目使用什么技术栈？

A: FastAPI + Tortoise ORM + PostgreSQL。详细说明见 [项目结构说明](./项目结构说明.md)。

### Q: 如何创建数据库迁移？

A: 运行 `aerich migrate --name your_migration_name`，然后 `aerich upgrade`。

### Q: 如何添加权限控制？

A: 在路由中添加 `dependencies=[DependAuth, DependPermission]`。

### Q: 如何避免 N+1 查询？

A: 使用 `prefetch_related()` 预加载关联数据。
